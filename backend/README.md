# Backend

The backend service receives sensor data from the ESP32 device, processes fall alerts, stores event information in MongoDB, and provides APIs for the monitoring dashboard.

## Responsibilities

* Receive fall alerts
* Store event data
* Manage API requests
* Communicate with database
* Serve dashboard data

## Technologies

* Node.js
* Express.js
* MongoDB
* Socket.IO

## Installation

```bash
npm install
```

## Start Server

```bash
npm start
```
